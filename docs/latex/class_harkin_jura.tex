\doxysubsection{Descrição detalhada}
Classe com as equacoes da isoterma de \mbox{\hyperlink{class_harkin_jura}{Harkin\+Jura}}  

Isoterma com dois parametros, $ K_1 $ e $ K_2 $, cujas formulas sao as seguintes\+: \textbackslash{}begin\{align\} \mbox{[}\textbackslash{}frac\{1\}\{Q\+\_\+e(\+C\+\_\+e, T)$^\wedge$\{2\}\} = \textbackslash{}frac\{K\+\_\+2 -\/ 1\}\{K\+\_\+1\}log(\+C\+\_\+e)\mbox{]} \textbackslash{}end\{align\} Um artigo de referencia pode ser encontrado \href{https://doi.org/10.1016/B978-0-12-804609-8.00005-4}{\texttt{ aqui}}. \begin{DoxyAuthor}{Autores}
