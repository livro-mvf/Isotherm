\doxysubsection{Descrição detalhada}
Classe com as equacoes da isoterma de \mbox{\hyperlink{class_halsey}{Halsey}}  

Isoterma com dois parametros, $ K_1 $ e $ K_2 $, cujas formulas sao as seguintes\+: \textbackslash{}begin\{align\} ln(\+Q\+\_\+e(\+C\+\_\+e)) = \textbackslash{}frac\{1\}\{K\+\_\+2\}ln\textbackslash{}left(\textbackslash{}frac\{K\+\_\+1\}\{C\+\_\+e\}\textbackslash{}right) \textbackslash{}end\{align\} Um artigo de referencia pode ser encontrado \href{https://doi.org/10.1016/B978-0-12-804609-8.00005-4}{\texttt{ aqui}}. \begin{DoxyAuthor}{Autores}
