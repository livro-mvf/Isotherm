\doxysubsection{Descrição detalhada}
Classe com as equacoes da isoterma de Brouers -\/ Sotolongo.  

Isoterma com tres parametros, $ q_{max} $, $ K_1 $ e $ K_2 $, cujas formulas sao as seguintes\+: \textbackslash{}begin\{align\} Q\+\_\+e(\+C\+\_\+e) = q\+\_\+\{max\} \textbackslash{}left\mbox{[}1-\/e$^\wedge$\{-\/K\+\_\+1\+C\+\_\+e$^\wedge$\{K\+\_\+2\}\}\textbackslash{}right\mbox{]} \textbackslash{}end\{align\} Um artigo de referencia pode ser encontrado \href{https://doi.org/10.1177/0263617416670909}{\texttt{ aqui}}. \begin{DoxyAuthor}{Autores}
Joao Flavio Vieira de Vasconcellos
