\doxysubsection{Descrição detalhada}
Classe com as equacoes da isoterma de \mbox{\hyperlink{class_valenzuela_myers}{Valenzuela\+Myers}}.  

Isoterma com tres parametros, $ q_{max} $, $ K_1 $ e $ K_2 $, cujas formulas sao as seguintes\+: \textbackslash{}begin\{align\} \mbox{[}Q\+\_\+e(\+C\+\_\+e, T)= \textbackslash{}frac\{q\+\_\+\{max\}\}\{2K\+\_\+2\}ln(\textbackslash{}frac\{K\+\_\+1+\+C\+\_\+ee$^\wedge$\{K\+\_\+2\}\}\{K\+\_\+1+\+C\+\_\+ee$^\wedge$\{-\/K\+\_\+2\}\})\mbox{]} \textbackslash{}end\{align\} Um artigo de referencia pode ser encontrado \href{https://doi.org/10.1016/B978-0-12-804609-8.00005-4}{\texttt{ aqui}}. \begin{DoxyAuthor}{Autores}
Joao Flavio Vieira de Vasconcellos
